Technologies Used by Yurik BV

  • Python: Yurik BV's team is fluent in Python, a popular and versatile programming language that is widely used in data science and machine learning. We use Python to develop AI and machine learning models, build data pipelines and APIs, and automate various processes and tasks.
  • Pandas and PySpark: Yurik BV uses Pandas and PySpark, two powerful Python libraries, to process and analyze large datasets. Pandas is great for working with small to medium-sized datasets, while PySpark is suitable for dealing with large and distributed datasets.
  • Gensim, FastText, PyTorch, and scikit-learn: Yurik BV uses a range of machine learning libraries and frameworks, such as Gensim, FastText, PyTorch, and scikit-learn, to build and train AI and machine learning models. These libraries and frameworks provide a wide range of algorithms and tools that can be used to solve a variety of business problems.
  • ElasticSearch and Neo4j: Yurik BV uses ElasticSearch and Neo4j, two popular database technologies, to store, query, and analyze complex and interconnected data. ElasticSearch is a powerful search engine that can be used to find relevant information in large datasets, while Neo4j is a graph database that is well-suited for modeling and analyzing complex relationships between data.
  • Docker and docker-compose: Yurik BV uses Docker and docker-compose to manage and deploy applications and services in a consistent and scalable way. Docker allows us to package applications and their dependencies into lightweight containers that can be easily deployed and run on any platform, while docker-compose makes it easy to define and manage complex environments and applications.
